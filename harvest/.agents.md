Agent Notes (Harvest Moon bot)

Single bot policy
- `harvest_bot.py` is the only bot entrypoint (human + autoplay).
- `run_bot.sh` wraps venv + launches `harvest_bot.py`.
- `farm_bot.py` has been removed; do not re‑introduce it.

Run commands
- Human mode: `./run_bot.sh play --state Y1_Spring_Day01_06h`
- Autoplay (auto‑clear): `./run_bot.sh clear --state Y1_Spring_Day01_06h --priority bushes --priority-only`
- Headless: `HEADLESS=1 ./run_bot.sh clear --state Y1_Spring_Day01_06h --priority bushes --priority-only`

Behavior
- Bot can be toggled in‑game via L+R+SELECT.
- Auto‑clear dismisses dialogs and avoids leaving the farm map.
