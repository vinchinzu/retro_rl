# ROM Setup

The ROM file for Harvest Moon (SNES) should be located at `../../roms/Harvest Moon.sfc`.

## Setup Instructions

1. Obtain a legal copy of Harvest Moon (SNES) ROM
2. Place it in the `roms/` directory as `Harvest Moon.sfc`
3. A symlink at `rom.sfc` points to this file for stable-retro compatibility

## File Requirements

- **Filename**: `Harvest Moon.sfc` or `Harvest Moon.smc`
- **Location**: `roms/` directory (gitignored)
- **SHA hash**: See `rom.sha` for verification

The `roms/` directory is gitignored to avoid copyright issues.
